 Банковское Приложение на Python

 Описание проекта

Это банковское приложение с графическим интерфейсом, реализованное на Python с использованием библиотеки Tkinter. Проект состоит из двух версий с различным функционалом и уровнем сложности.

 Особенности

 Версия 1.0 - Базовая
- Регистрация новых пользователей
- Авторизация в системе
- Основные банковские операции:
  - Пополнение счета
  - Снятие наличных
  - Переводы между пользователями
  - Просмотр истории операций
- Простой и интуитивный интерфейс
- Хранение данных в JSON-формате

 Версия 2.0 - Расширенная
- Все функции версии 1.0
- Два новых модуля:
  1. Модуль накоплений - накопительный счет с процентами
  2. Модуль кредитования - кредитный калькулятор
- Улучшенная валидация данных
- Современный интерфейс
- Повышенная безопасность
- Гостевой режим
- Фильтрация и экспорт истории операций
- Система блокировок при неудачных попытках входа


 Требования

- Python 3.6 или выше
- Стандартные библиотеки Python:
  - tkinter
  - json
  - hashlib
  - os
  - re
  - datetime

 Как использовать

Запуск версии 1.0:
```bash
python bank1.0.py
```

 Запуск версии 2.0:
```bash
python bank2.0.py
```

  Функционал

 Общий для обеих версий:
1. Регистрация - создание нового банковского аккаунта
2. Авторизация - вход в систему
3. Управление счетом - просмотр баланса
4. Финансовые операции - пополнение, снятие, переводы
5. История операций - просмотр всех транзакций

 Только в версии 2.0:
1. Накопительный счет - отдельный счет с начислением процентов
2. Кредитный калькулятор - расчет платежей по кредиту
3. Расширенная безопасность - блокировка после 5 неудачных попыток
4. Валидация данных - проверка email, пароля, телефона
5. Экспорт данных - сохранение истории в файл

 Безопасность

- Пароли хранятся в хешированном виде
- Проверка сложности паролей
- Защита от повторной регистрации
- Ограничение неудачных попыток входа (v2)
- Валидация всех входящих данных

 Интерфейс

 Версия 1.0:
- Простой и понятный интерфейс
- Минималистичный дизайн
- Базовая цветовая схема

 Версия 2.0:
- Современный дизайн
- Адаптивная верстка
- Цветовая схема Material Design
- Карточки для отображения информации
- Анимации и улучшенный UX

Хранение данных

Все данные хранятся в JSON-файлах:
- users.json (v1) / bank_users_v2.json (v2) - информация о пользователях
- transactions_v2.json (только v2) - история всех операций
